#### _Technologies:_
- Java 11
- Spring Boot (MVC, Security, REST)
- Spring DATA
- Hibernate + h2
- JUnit (Mockito)
- Lombok
- Thymeleaf
- Security: JWT-token
- Front-end: Angular 15.1.6, rxjs 7.8.0, typescript 4.9.5
